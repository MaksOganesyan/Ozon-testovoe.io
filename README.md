# Progress — круговой индикатор для Ozon (тестовое)

Сделал прототип блока прогресса, как в ТЗ: круглый индикатор для мобильного веба. Показывает процент выполнения, умеет анимироваться (крутится) и прятаться.

Демо:

## Что внутри

- Круг начинается сверху (12 часов), заполняется по часовой стрелке
- Value 0–100 — меняет длину дуги
- Animate — заставляет круг крутиться постоянно (~2 сек оборот)
- Hide — прячет индикатор (visibility: hidden), контролы остаются
- Всё независимо: можно крутить анимацию при 0%, прятать при 50% и т.д.

Управление — прямо на странице: инпут для значения + два свитча.

Например через расширение Live Server
