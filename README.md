# Progress — круговой индикатор для Ozon тестовое

Блок прогресса, круглый индикатор. Показывает процент выполнения, умеет крутится и прятаться.

Демо: https://maksoganesyan.github.io/Ozon-testovoe.io

## Что внутри

- Круг начинается сверху (12 часов), заполняется по часовой стрелке
- Value 0–100 — меняет длину дуги
- Animate — заставляет круг крутиться постоянно 
- Hide — прячет индикатор (visibility: hidden), контролы остаются
- Всё независимо: можно крутить анимацию при 0%, прятать при 50% и т.д.

Управление — прямо на странице: инпут для значения + два свитча. 
Или через Api как показано ниже

## API

Api сделан через `ProgressAPI`:

```javascript
ProgressAPI.setValue(50);      
ProgressAPI.setAnimated(true); 
ProgressAPI.setHidden(true);
ProgressAPI.getState();       
```
